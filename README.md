# bitonic-sort.ts

🪄🪄🪄 バイトニックソートをTypeScriptで実装してみる！！！  

## 環境構築

### 通常の方法

```shell
# 必要なパッケージのインストール
yarn install

# デバグ実行
yarn dev

# ビルド
yarn build

# ビルド後の実行
yarn start
```

### Dockerを使う方法

```shell
# Dockerイメージのビルド
docker build -t bitonic-sort .

# Dockerコンテナの起動
docker run -it --rm bitonic-sort
```
